O conhecimento é como uma janela que se abre para o mundo!
==========================================================

> É notável a dificuldade que muitos profissionais encontram para admitir que são falíveis e que não possuem todo o conhecimento disponível sobre qualquer assunto.
>
>O conhecimento deve ser encarado da seguinte forma: Você sabe um pouco e todas as outras pessoas no mundo sabem o resto. Portanto, mais importante do que tentar transmitir uma imagem de sabedoria é saber onde conseguir o conhecimento que lhe falta.
>
>Diante de qualquer dificuldade, primeiramente devemos nos perguntar: O que me falta saber para que esta tarefa seja executada, para que este mecanismo funcione, para que os resultados das vendas melhorem, para que meu site receba mais visitas..
